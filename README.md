# Grassmann time evolving matrix product operator (GTEMPO) method for fermionic impurity problem 
## This repo implements the algorithms in the following list of papers
- PRB 109, 045140 (2024) for the generic framework of GTEMPO and calculations on the real-time axis 
- NJP 26, 013109 (2024) for GTEMPO calculations on the imaginary-time axis 
- PRB 110, 165114 (2024) for GTEMPO calculations on the L-shaped Kadanoff-Baym axis 
- SciPost Phys. Core 7, 063 (2024) for the translational invariant approach to build the Grassmann MPS representation of the Feynman-Vernon influence functional
